# ELRS协议接收机产品介绍

## 产品概述

ELRS (ExpressLRS) 是一款开源的高性能无线电控制链路协议，专为FPV无人机和遥控模型设计。本接收机基于ELRS协议开发，提供超低延迟、高可靠性和长距离的无线控制解决方案。

## 核心特性

### 协议优势

- **超低延迟**：端到端延迟可低至5ms，提供近乎实时的操控体验
- **长距离传输**：支持数公里的控制距离，满足各种飞行场景需求
- **高刷新率**：支持最高500Hz的刷新率，确保精准控制
- **抗干扰能力强**：采用FHSS跳频技术，有效抵抗环境干扰
- **开源免费**：完全开源协议，社区活跃，持续更新优化

### 硬件特性

- **轻量化设计**：重量轻，适合各类无人机和模型
- **低功耗**：优化的电源管理，延长使用时间
- **多接口支持**：支持SBUS、CRSF、IBUS等多种输出协议
- **即插即用**：简化配置流程，快速上手
- **固件可升级**：支持在线固件更新，保持最新功能

## 配置与使用

### 入门四步走

#### 步骤1：启用高频头

进入遥控器模型设置 → SETUP 页面 → 开启 External RF Mode → Mode 设为 CRSF，Baudrate 设为 1.87M

> **重要**
>
> 波特率建议设为 **1.87M**，否则可能无法进入 ELRS Lua 脚本或无法设置 F1000 速率。

#### 步骤2：高频头与接收机对频

- **方法一**：三连上电对频（接收机连续上电三次进入对频模式）
- **方法二**：WiFi 配置对频密码（电脑连接接收机 WiFi 修改密码）

#### 步骤3：固件升级

通过  APP 工具栏中的「Sirius 升级」功能，一键升级高频头、接收机和 Backpack 固件。

#### 步骤4：模式配置

根据需求选择 CRSF 或 MAVLink 模式，并在 ELRS Lua 脚本中进行相应配置。

### 硬件连接

#### 接线方式

接收机通常有以下接口：

| 接口 | 功能         | 连接对象               |
| ---- | ------------ | ---------------------- |
| VCC  | 电源正极     | 飞控5V输出或BEC        |
| GND  | 电源负极     | 飞控GND                |
| RX   | 串行数据输入 | 飞控TX（用于固件升级） |
| TX   | 串行数据输出 | 飞控RX（用于信号输出） |
| SBUS | SBUS信号输出 | 飞控SBUS输入           |

#### 连接示例

**SBUS输出模式**：

```
接收机VCC → 飞控5V
接收机GND → 飞控GND
接收机SBUS → 飞控SBUS_IN
```

**CRSF输出模式**：

```
接收机VCC → 飞控5V
接收机GND → 飞控GND
接收机TX → 飞控RX（指定UART）
```

### 固件升级

#### 准备工作

1. 下载并安装 [ELRS Configurator](https://github.com/ExpressLRS/ExpressLRS-Configurator)
2. 准备USB转TTL适配器（如CP2102、FT232）
3. 确保接收机与电脑连接

#### 升级步骤

1. 打开ELRS Configurator，选择正确的端口和波特率
2. 进入"Flashing"选项卡
3. 选择目标接收机型号和固件版本
4. 点击"Flash"开始升级
5. 等待升级完成，设备自动重启

#### 无线升级（OTA）

1. 确保发射机和接收机已绑定
2. 在发射机菜单中选择"ELRS OTA Update"
3. 选择要升级的固件文件
4. 等待升级完成（约1-2分钟）

### 发射机绑定

#### 方法一：按键绑定

1. 接收机断电
2. 按住接收机绑定按钮不放
3. 给接收机上电，LED开始快闪表示进入绑定模式
4. 在发射机中进入"Bind"菜单
5. 选择对应的协议和频段
6. 等待绑定成功（LED常亮或慢闪）

#### 方法二：CLI命令绑定

```bash
# 进入Betaflight CLI
bind_rx
# 接收机进入绑定模式后发送
bind
```


### 参数调节

#### 刷新率设置

| 刷新率 | 适用场景   | 延迟  |
| ------ | ---------- | ----- |
| 25Hz   | 低功耗模式 | ~50ms |
| 50Hz   | 常规飞行   | ~25ms |
| 100Hz  | FPV竞速    | ~12ms |
| 200Hz  | 高要求竞速 | ~7ms  |
| 500Hz  | 极致性能   | ~5ms  |

#### 功率调节

- **低功率（10-50mW）**：室内飞行、近距离操控
- **中功率（100mW）**：常规户外飞行
- **高功率（250mW）**：长距离飞行

#### Telemetry配置

在ELRS Configurator中启用Telemetry后，可获取：

- 接收机电压
- RSSI信号强度
- 链路质量
- 电池状态（需飞控支持）

### 实际使用技巧

#### 天线摆放

- 天线应远离金属物体和电机
- 双天线版本建议天线夹角为90度
- 避免天线被遮挡

#### 信号检测

- 飞行前检查RSSI值（建议> -80dBm）
- 注意信号丢失警告
- 定期检查电池电压

#### 常见故障排查

| 现象         | 可能原因     | 解决方法       |
| ------------ | ------------ | -------------- |
| 接收机不响应 | 未绑定       | 重新绑定       |
| 信号不稳定   | 天线问题     | 检查天线连接   |
| 通道反向     | 通道映射错误 | 调整飞控配置   |
| 失控保护触发 | 信号丢失     | 检查距离和干扰 |
