# mLRS 快速上手指南

欢迎使用 mLRS（micro Long Range System）！本指南是所有文档的总览入口，帮助您快速了解 mLRS 系统并导航到详细文档。

---

## 什么是 mLRS

mLRS 是一款**"数遥一体"**的开源遥控协议，将遥控功能和数传功能整合到同一条链路中。

### 核心特点

| 特点 | 说明 |
|------|------|
| **数遥一体** | 遥控和数传共用一条链路，设备更精简 |
| **双向通信** | 同时支持控制指令下发和遥测数据回传 |
| **蓝牙输出** | 高频头内置蓝牙，可直接连接手机 APP |
| **参数丰富** | 支持完整的 MAVLink 遥测数据传输 |

## 常见问题（FAQ）

### Q1：Tx 和 Rx 无法绑定怎么办？

**A**：请按照以下步骤排查：
1. 确保两者都已上电并进入绑定模式
2. 检查电源是否稳定（推荐使用 5V 2A 以上电源）
3. 确保固件版本一致
4. 尝试更换绑定 phrase（默认是 'mlrs.0'）

### Q2：为什么我看不到遥测数据？

**A**：请检查以下设置：
1. 确认 `Rx Snd RadioStat` 参数设置为 `ardu_1` 或 `meth_b`
2. 确认飞控串口配置正确（MAV_x_RADIO = 1）
3. 检查串口接线是否正确（交叉连接：Tx→RX，RX→Tx）
4. 确认波特率设置匹配

### Q3：RC 通道没有信号怎么办？

**A**：可能的原因：
1. 遥控器协议设置错误（应设置为 CRSF，波特率 400K）
2. `Tx Ch Source` 参数未设置为 `crsf`
3. 接收器输出模式未正确配置
4. 飞控 RC 协议未设置为 CRSF（RC_PROTOCOLS = 1）

### Q4：如何恢复出厂设置？

**A**：在终端中执行以下命令：
```bash
p factory_reset;
pstore;
reload;
```

### Q5：为什么参数修改后重启就失效了？

**A**：修改参数后必须执行 `pstore;` 命令保存配置，否则重启后会恢复到之前的设置。

### Q6：可以使用哪些频段？

**A**：mLRS 支持多个频段：
- **868 MHz**：欧洲常用频段
- **915 MHz**：北美常用频段
- **2.4 GHz**：全球通用频段（与 WiFi 同频段，可能有干扰）

### Q7：如何选择合适的功率设置？

**A**：根据飞行距离和法规要求选择：
- **低功率（10-100 mW）**：近距离飞行，合规性好
- **中功率（250-500 mW）**：中等距离，适合大多数场景
- **高功率（1W+）**：远距离飞行，需注意当地法规限制

### Q8：地面站连接失败怎么办？

**A**：请检查：
1. 串口选择是否正确
2. 波特率设置是否匹配
3. 飞控 MAVLink 协议是否设置为 v2（MAV_x_PROTOCOL = 2）
4. 确保没有其他软件占用该串口
