## 1. 终端输入指令修改 mLRS 参数

### 所需材料

| 材料           | 说明                     | 推荐型号  |
| -------------- | ------------------------ | --------- |
| 内置USB-TTL 适配器高频头 | 用于连接 mLRS 模块到电脑 | CH340     |
| type-c 线      | 用于连接适配器和模块     | type-c 线 |
| nano接口遥控器         | 给 mLRS 模块供电         | 遥控器    |
| 串口终端软件   | 用于发送指令             | VOFA+     |

### 准备工作

- **连接方式**：

  1. 将高频头连接遥控器，拆下高频头后盖，找到高频头模块

  ![1778205363952](image/mLRS查看及更改参数/1778205363952.png)

  2. 找到"Bind"按钮，按住它，给遥控器上电，上电后2秒黄灯慢闪可松开按钮（注意:遥控器警告时无法给高频头模块供电，指示灯无法正常亮起！！）

![1778205777663](image/mLRS查看及更改参数/1778205777663.png)

  ![1778205748646](image/mLRS查看及更改参数/1778205748646.png 

 3. 连接 USB-TTL 适配器前需下载[CH340驱动程序](https://espjs.cn/driver)

![1778206669712](image/mLRS查看及更改参数/1778206669712.png))

4. 通过 TYPE-C 端口连接 USB-TTL 适配器，将 USB-TTL 适配器连接到电脑，打开 [VOFA+ 串口终端软件](https://www.vofa.plus/downloads/)

  ![1778206305281](image/mLRS查看及更改参数/1778206305281.png)



点击连接按钮

![1778207382532](image/mLRS查看及更改参数/1778207382532.png)

5. 串口连接参数：端口号选择新出现的端口号，波特率 115200 bps，8 数据位，无校验位，1 停止位（简称 **8N1**）（与天空端对频后，可将天空端上电对天空端远程调参）

![1778148952132](image/mLRS查看及更改参数/1778148952132.png)

* 若未能成功连接高频头模块，在VOFA+串口调试软件中显示可以成功连接但发送指令无法成功执行

![1778209439050](image/mLRS查看及更改参数/1778209439050.png)

> **新手提示**：TX 和 RX 是交叉连接的（适配器 TX → 模块 RX，适配器 RX → 模块 TX），这是串口通信的基本规则。

### 常用指令

- **查看所有参数**：`pl;`
- **查看特定参数**：`p 参数名;`（参数名中的空格用下划线替代，例如 `p tx_power;`）
- **修改参数**：`p 参数名 = 值;`（例如 `p tx_power = 0;`）
- **保存参数**：`pstore;`（参数修改后必须执行此命令才能永久保存）
- **查看命令帮助**：`h;` 或 `help;` 或 `?;`
- **查看版本信息**：`v;`
- **重新加载参数**：`reload;`
- **开始绑定**：`bind;`
